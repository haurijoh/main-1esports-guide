# main-1esports-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容主要围绕电子竞技信息整理与索引，不针对任何特定网站或域名。

## 项目简介

这是一个非商业化的个人归档项目，旨在集中存放并公开一些独立编写的 HTML 页面。每个页面通常包含特定主题下的资源整理、信息导航或说明文档，方便通过浏览器直接访问。

## 目录结构

```
├── index.html          # 首页 / 导航页面
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/             # 页面引用的静态资源（如有）
    ├── css/
    ├── js/
    └── images/
```

- `index.html`：项目入口页面，列出所有归档页面的链接。
- `pages/`：主要归档目录，每个文件为一个独立完整的 HTML 页面。
- `assets/`：可选，存放页面所需的样式、脚本或图片文件。

## 页面归档说明

- 所有页面均为静态 HTML，无需后端服务即可通过浏览器直接打开。
- 页面内容以信息整理和索引为主，不包含动态功能或外部依赖。
- 页面之间无强关联，可独立浏览。
- 内容可能不定期更新或调整，不保证长期一致性。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/main-1esports-guide.git
   ```
2. 直接在浏览器中打开 `index.html` 或 `pages/` 目录下的任意页面。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后在线访问。
